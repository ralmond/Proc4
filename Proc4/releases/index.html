This web site will eventuall host three projects related to the infrastructure of Physics Playground.

Proc4

This generally handles messages from the four process architecture. It also handles logging to databases and recovery of messages from databases.

Code is not yet released, but the subverions repository is at: https://pluto.coe.fsu.edu/svn/common/Proc4/

EIEvent

This is a rule based system for processing event logs to produce observables. A draft document of the rule format and logic is available.

Code is not yet released, but the subverions repository is at: https://pluto.coe.fsu.edu/svn/common/EIEvent/

EABN

This handles evidence accumulation using Bayesian networks. It uses RNetica for its Bayes net engine.

Code is not yet released, but the subverions repository is at: https://pluto.coe.fsu.edu/svn/common/EABN/
